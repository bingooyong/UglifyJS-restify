UglifyJS-restify
================

UglifyJS-restify


安装uglify-js
============

	npm install uglify-js
	


安装node-restify
===============

使用[node-restify](https://github.com/mcavage/node-restify)搭建restful服务

	npm install restify	

运行restify
==========

	export NODE_PATH=/app/nodejs/ecsmall/node_modules 

	/app/nodejs/ecsmall/bin/node /app/nodejs/test/restify.js
